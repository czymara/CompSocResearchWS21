a) Macro-level data

Let us add the country-level data. There are two sources: macrodata_1.xlsx and macrodata_2.csv. Import the XLSX using the read_excel() function of the readxl package and assign it to an object macrodata_1. Import the CSV and assign it to macrodata_2. Which countries are included in the ESS but not in macrodata_1 and not in macrodata_2, respectively?

c) Joining data

Combine the ESS with both macro-level data sets using one of the join functions from the dplyr package.

data_combined <- left_join(ESS16, macrodata_1, by = "cntry") %>%
  left_join(., macrodata_2, by = "cntry")

c) Plotting

For each macro-variable, create a scatter plot with average life satisfaction on the y-axis and the economic indicator on the x-axis (use ggplot2()).

# GDP
data_combined %>%
  ggplot(aes(x = c_pps_habeu_2016,
             y = stflife,
             label = cntry)) +
  stat_summary(fun = mean,
               geom = "text") +
  geom_smooth(method = "lm") +
  ylab("Average life satisfaction") +
  xlab("GDP/head")
