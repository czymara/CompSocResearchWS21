3. Random slope model

a) Varying effect of political orientation

We want to know whether the relationship between welfare state attitudes and political orientation differs across countries. Estimate a model including a random slope for the (centered) political orientation variable:

\[sbeqsoc_{ij}=\beta_0+\beta_1lrscale\_cent_{ij}+u_{0j}+u_{1j}lrscale\_cent_{ij}+e_{ij}\]

Interpret the coefficient of the fixed (\(\beta_1\)) and of the random effect (\(\tau_{11}=var(u_{1j})\)) of political orientation.

RSlope <- lmer(sbeqsoc ~ lrscale_cent + (1 + lrscale_cent | cntry), data = ESS8) # unstructured covariance matrix

c) Random effects

Use ranef() to access the random intercepts and slopes for each country (add condVar = TRUE to take covariance matrix into account). Can you name a country where people were rather negative about the welfare state but where the expected effect of political orientation is stronger than average? What about a country where welfare views are rather positive but the expected effect of political orientation is weaker?

randomEffects <- ranef(RSlope, condVar = TRUE) 

# inspect random effects for each country
randomEffects_df <- as.data.frame(ranef(RSlope, condVar = TRUE)$cntry) # as data frame
randomEffects_df$cntry <- rownames(randomEffects_df) # add country names variable
