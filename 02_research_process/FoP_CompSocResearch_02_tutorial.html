3. Plotting

Letâs plot this finding. What we want is a bar chart with the mean values of life satisfaction for each religious group. For this, we need the tidyverse, as we will work with its ggplot2-package. ggplot2 might be the most powerful tool for data vizualization, as it can plot basically everything. The basis for this is the initial plot, where you define the data, x and y in the following way: ggplot(data, aes(x, y)). Adapt this code to plot rlgdnm_lbl as our x and stflife as y. If you need help, check https://ggplot2.tidyverse.org/ or https://www.r-graph-gallery.com/ggplot2-package.html

# load tidyverse
if(!require(tidyverse)){
    install.packages("tidyverse")
    library(tidyverse)
}

ggplot(ESS02, aes(x = rlgdnm_lbl,
           y = stflife))

You will see that this will create a two-dimensional plot, with an x-axis and a y-axis However, it wonât show any data. This is because we havenât told ggplot2 yet how to display the data (e.g., density plot, line chart, scatter plot, histogram etc.). What we want here is a bar chart, so we add a new layer (+) to our plot: geom_bar() (see here). By default, geom_bar() will show the number of cases in each group of x. However, we want the mean values of y for each group of x. This can be achieved using geom_bar(stat = "summary", fun = "mean"). Create the bar chart showing mean values of life satisfaction by religion.

# Distribution of life satisfaction
ggplot(ESS02, aes(x = rlgdnm_lbl,
           y = stflife)) +
  geom_bar(stat = "summary",
           fun = "mean")

# Nicer:
ESS02 %>%
  drop_na(stflife) %>% # need to drop missings for ordering on y
  ggplot(aes(x =  reorder(rlgdnm_lbl, stflife, mean), # order on y
           y = stflife)) +
  geom_bar(stat = "summary",
           fun = "mean") +
  labs(y = "Average lifesatisfaction", x = "Religious denomination") +
  theme(axis.text.x = element_text(angle = 45,
                                   hjust=1))

b) Plot

Letâs plot this. We will again use ggplot2. This time we will not plot x against y, but we want a so-called stacked bar chart, where the full bar consists of the percentages that are stacked on top of each other (summing up to 1, see Percent stacked barchart). Hence, we have to change the code to: ggplot(aes(x = cntry, fill = rlgdnm_lbl)). In addition, we have to change the code for the bar chart to: geom_bar(position="fill"). Create the bar chart with the religious groups stacked per country.

# stacked bar chart
ESS02 %>%
  ggplot(aes(x = cntry,
           fill = rlgdnm_lbl)) +
  geom_bar(position="fill") +
  labs(y = "Proportions", x = "", fill = "Religious denomination") +
  theme(axis.text.x = element_text(angle = 45,
                                   hjust=1))
